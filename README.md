# RaiderAid-Tutorial-Management
This repository contains the files and programs for the Email Reminder, a tutorial management tool that lets teachers set up notifications for student groups through email.

## Live App
GitHub Pages URL:
`https://jonathan-joy1.github.io/RaiderAid-Tutorial-Management/`

## Install As PWA
1. Open the live app URL in Chrome or Edge.
2. Click the install icon in the address bar (or use browser menu -> Install app).
3. Launch RaiderAid from your installed apps list.

## Quick Usage
1. Sign up or log in with a verified email.
2. Add learners in `Learner List`.
3. Send reminders from `Sender`.
4. Learners confirm attendance from the email link.

## Email Setup (EmailJS and Firebase)
The processes of email verification for signing up and sending emails to students involve EmailJS and Firebase.

## Firebase Config
Firebase configuration will be ignored and not staged.
